# Sistema Inteligente de Priorização de Processos Jurídicos

## Descrição

Projeto de Machine Learning desenvolvido para auxiliar escritórios de advocacia na priorização automática de processos jurídicos.

O sistema classifica processos em:

- Alta prioridade
- Média prioridade
- Baixa prioridade

com base em fatores estratégicos como:

- valor da causa;
- urgência;
- prazo processual;
- quantidade de partes;
- tempo sem movimentação.

## Tecnologias Utilizadas

- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn
- Google Colab

## Funcionalidades

- análise exploratória de dados;
- transformação de variáveis;
- treinamento de modelo;
- matriz de confusão;
- previsão automática.

## Objetivo

Demonstrar a aplicação prática de Machine Learning na gestão jurídica.
